OOPS Banner App

Project Description

This project is a Java-based application that displays the word "OOPS" in a banner format using "*" characters.
It demonstrates core Object-Oriented Programming (OOP) concepts and structured programming techniques.

---

Objectives

- Understand and implement Java basics
- Practice Object-Oriented Programming
- Learn array handling and loops
- Improve code modularity and reusability

---

Features

- Displays "OOPS" banner
- Uses "*" characters for pattern printing
- Multiple implementations using:
  - Basic print statements (UC1)
  - "String.join()" (UC2)
  - Arrays and loops (UC3)
  - Method-based design (UC5)
  - Class-based pattern mapping (UC6)
- Clean and structured code

How to Run

1. Clone the repository:
   
   git clone https://github.com/as1397-AareenSingh/OOPS_BANNER_APP.git

2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)

3. Compile and run:
   
   javac oopsbannerapp.java
java oopsbannerapp
